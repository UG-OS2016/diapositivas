Kernel

Se puede definir como el corazón del sistema operativo. Es el encargado de que el software y el hardware del equipo trabajen juntos.

Las funciones más importantes de este aunque no las únicas, son:

  • Administración de la memoria para todos los programas y procesos en ejecución.
  • Administración del tiempo de procesador que los programas y procesos en ejecución utilizan.
  • Es el encargado de que podamos acceder a los periféricos/elementos de nuestro ordenador de una manera cómoda.

Biblioteca estándar de procedimientos

Son un conjunto de implementaciones funcionales las cuales se encargan de permitir al usuario comunicar el software con el kernel, La mayoría de los sistemas operativos Linux, poseen la biblioteca llamada libc.

Por ejemplo, las llamadas al kernel para abrir un archivo, para lectura o escritura, o mostrar texto en pantalla, o leer valores desde el teclado.
